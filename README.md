# 📚 Article Hub

A modern, beautifully designed web platform for reading and publishing articles across various categories, featuring an interactive UI and dynamic theme switching.

Live Demo: (https://bid0nchik.github.io/SDiaPaCWfPAotInfGaD.github.io/)

---

## ✨ Features

*   **Categorized Reading** – Explore a wide range of articles organized into distinct sections and topics.
*   **Role-Based Content Creation** – Authorized users can write, edit, and publish articles directly through an intuitive built-in editor.
*   **Dynamic Theme Switching** – Toggle effortlessly between Light and Dark modes to reduce eye strain.
*   **Stunning Responsive Design** – Modern, clean, and pixel-perfect UI tailored for both desktop and mobile devices.

---

## 🛠️ Technologies Used

*   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
*   **Backend & Server:** Node.js / Docker
*   **Deployment:** Render

---

## 🔒 Access & Permissions

*   **Guests / Readers:** Can browse, filter sections, read articles, and switch UI themes.
*   **Authors / Admins:** Granted access to the publishing panel to create and manage articles.
